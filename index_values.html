Index of values


C
create [Index]
Index.create creates a new, empty index.

D
documents [Index]
Index.documents i returns the number of indexed documents in i.
documents_similarity [Index]
Index.documents_similarity doc1 doc2 index calculates similarities between documents doc1 and doc2 using index index.

F
from_directory [Index]
Index.from_directory dir creates index from all files from dir, including all files in dir subdirectories.

I
indexed_documents [Index]
Index.indexed_documents i returns list of all documents indexed in i.

L
load_from_file [Index]
Index.load_from_file file returns index loaded from file named name.

M
more_like_this [Index]
Index.more_like_this doc i returns list of tuples (document, similarity) for document doc and index with term vectors (Index.tvector) i sorted descending by similarity to doc.

S
save_to_file [Index]
Index.save_to_file i name saves index i to file named name.
similarity [Index]
Index.similarity v1 v2 calculates similarity between term vectors v1 and v2.
stem [Stemmer]
Stemmer.stem s returns stem for s.

T
term_vector [Index]
Index.term_vector doc i returns Index.tvector for doc from Index.tvector index i.
